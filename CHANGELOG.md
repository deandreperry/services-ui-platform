# Changelog

## 0.1.0

- Initial platform scaffold with pnpm workspaces, Turborepo, TypeScript, React, Vite, Storybook, Vitest, ESLint, Prettier, Changesets, and GitHub Actions.
- Added design token package with typed token exports, CSS variables, and light/dark themes.
- Added accessible React component library with Button, Card, Badge, TextField, Checkbox, Tabs, Modal, Toast, and Tooltip.
- Added accessibility utility package and icon package.
- Added documentation and playground apps that consume the platform packages.
